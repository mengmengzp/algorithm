## 做题方法论

1. 切碎知识点
2. 刻意练习
3. 反馈

### 5遍刷题法

#### 刷题第一遍
1. 5-10分钟：读题和思考
2. 直接看解法：多解法、比较解法优劣
3. 背诵、默写好的解法

#### 刷题第二遍

1. 自己写，然后leetcode提交
2. 多种解法比较、体会—-> 优化！

#### 刷题第三、四、五遍
间隔一天/一周/面试前
